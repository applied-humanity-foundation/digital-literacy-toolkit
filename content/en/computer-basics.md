# Computer Basics

## Understanding Your Computer

A computer is a machine that processes information. Whether you use a desktop, laptop, or tablet, the core principles are the same. Every computer has a processor (CPU) that performs calculations, memory (RAM) that holds data being actively used, and storage (hard drive or SSD) for permanent data.

The operating system — Windows, macOS, or Linux — is software that manages your hardware and provides the interface you interact with daily. Think of it as the foundation on which all your applications run.

## The Desktop Environment

Your desktop is the main screen you see after logging in. It typically includes icons for applications, a taskbar (Windows) or dock (macOS) for quick access, and a system tray showing the clock, network status, and notifications.

You can customize your workspace by organizing icons into groups, changing the wallpaper, and pinning your most-used apps to the taskbar for one-click access.

## File Management

Files are stored in folders, forming a tree-like hierarchy. Your computer usually comes with default folders like Documents, Downloads, Pictures, and Desktop. Creating a logical folder structure — for example, organizing by project or date — saves enormous time later.

Common file types you'll encounter:
- **Documents**: .docx, .pdf, .txt
- **Images**: .jpg, .png, .svg
- **Videos**: .mp4, .mov
- **Archives**: .zip, .rar

## Keyboard Shortcuts

Mastering keyboard shortcuts will dramatically speed up your work:

| Shortcut | Action |
|----------|--------|
| Ctrl+C | Copy |
| Ctrl+V | Paste |
| Ctrl+Z | Undo |
| Ctrl+S | Save |
| Ctrl+F | Find |
| Alt+Tab | Switch windows |

On macOS, replace Ctrl with the Command (⌘) key.
