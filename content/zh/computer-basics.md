# 计算机基础

## 认识你的计算机

计算机是一种处理信息的机器。无论你使用台式机、笔记本还是平板电脑，其核心原理都是相同的。每台计算机都有处理器（CPU）负责运算，内存（RAM）存放正在使用的数据，以及硬盘或固态硬盘（SSD）用于永久存储数据。

操作系统——Windows、macOS 或 Linux——是管理硬件并提供用户界面的软件，你可以把它理解为所有应用程序运行的基础平台。

## 桌面环境

桌面是你登录后看到的主屏幕。通常包括应用程序图标、任务栏（Windows）或程序坞（macOS）用于快速启动程序，以及系统托盘区域显示时间、网络状态和通知。

你可以通过整理图标分组、更换壁纸、将常用应用固定到任务栏等方式来个性化你的工作空间。

## 文件管理

文件存储在文件夹中，形成树状层级结构。你的计算机通常自带文档、下载、图片、桌面等默认文件夹。建立合理的文件夹结构——比如按项目或日期分类——能在日后节省大量时间。

常见文件类型：
- **文档**：.docx、.pdf、.txt
- **图片**：.jpg、.png、.svg
- **视频**：.mp4、.mov
- **压缩包**：.zip、.rar

## 键盘快捷键

掌握键盘快捷键能显著提高你的工作效率：

| 快捷键 | 功能 |
|--------|------|
| Ctrl+C | 复制 |
| Ctrl+V | 粘贴 |
| Ctrl+Z | 撤销 |
| Ctrl+S | 保存 |
| Ctrl+F | 查找 |
| Alt+Tab | 切换窗口 |

在 macOS 上，将 Ctrl 替换为 Command（⌘）键。
