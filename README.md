## About

The Digital Literacy Toolkit is a free, open-source collection of interactive learning modules designed to make essential technology skills accessible to everyone. Built as part of the Applied Humanity Foundation's mission to advance digital literacy and human well-being through responsible technology.

## Features

- **6 Learning Modules** — From computer basics to AI fundamentals
- **Interactive Quizzes** — Test understanding with built-in assessments
- **Progress Tracking** — Pick up where you left off with local progress saving
- **Multilingual** — English and Chinese (more languages welcome)
- **No Account Required** — Start learning immediately, no sign-up needed
- **Fully Offline** — Download and use without internet connection
- **Accessible** — Designed for learners of all backgrounds and skill levels

## Modules

| Module | Level | Description |
|--------|-------|-------------|
| Computer Basics | Beginner | File management, OS navigation, keyboard shortcuts |
| Internet Safety | Beginner | Passwords, phishing awareness, privacy protection |
| Productivity Tools | Beginner | Documents, spreadsheets, cloud collaboration |
| AI Fundamentals | Intermediate | How AI works, LLMs, prompt engineering basics |
| Data Literacy | Intermediate | Reading charts, basic statistics, data visualization |
| Digital Communication | Beginner | Email etiquette, video calls, online collaboration |

## Getting Started

```bash
# Clone the repository
git clone https://github.com/applied-humanity-foundation/digital-literacy-toolkit.git

# Open in your browser
open index.html
```

No build tools, frameworks, or dependencies required. Just open `index.html` in any modern browser.

## Contributing

We welcome contributions from educators, developers, translators, and learners worldwide. Here's how you can help:
